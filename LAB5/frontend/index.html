JavaScript вимкнено

Для роботи додатку необхідно увімкнути JavaScript у вашому браузері.

Як увімкнути JavaScript:

  • Google Chrome: Налаштування → Конфіденційність та безпека → Налаштування сайтів → JavaScript → Дозволити
  • Mozilla Firefox: Налаштування → Конфіденційність та захист → Захист від стеження → Налаштування → JavaScript → Дозволити
  • Microsoft Edge: Налаштування → Cookies та дозволи сайтів → JavaScript → Дозволити
  • Safari: Налаштування → Безпека → JavaScript → Дозволити

Після увімкнення JavaScript, будь ласка, перезавантажте сторінку.
